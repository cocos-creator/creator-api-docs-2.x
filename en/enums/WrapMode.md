### `WrapMode` Enum



Module: [cc](../modules/cc.md)


Specifies how time is treated when it is outside of the keyframe range of an Animation.


### Index
  - `Default`
  - `Normal`
  - `Reverse`
  - `Loop`
  - `LoopReverse`
  - `PingPong`
  - `PingPongReverse`

### Details

> Reads the default wrap mode set higher up.
