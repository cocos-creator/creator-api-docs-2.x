### 索引

##### 属性（properties）

  - [`ONE`](#one) `Vec3` 新 Vec3 对象。
  - [`ZERO`](#zero) `Vec3` 返回 x = 0，y = 0，z = 0 的 Vec3 对象。
  - [`UP`](#up) `Vec3` 返回 x = 0, y = 1, z = 0 的 Vec3 对象。
  - [`RIGHT`](#right) `Vec3` 返回 x = 1，y = 0，z = 0 的 Vec3 对象。
